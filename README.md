### Changes
- Playback occurs 2 seconds after hitting play.
- The stop playback key is changed to `Pause`.

### How to compile
- For beginner: Fork this repo, push any change, and GitHub Actions will automatically put the .exe in `Releases` (takes ~2 minutes).
- For veteran: You should know how to compile already.
